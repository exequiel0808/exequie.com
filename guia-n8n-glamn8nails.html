Automatización con n8n

Conecta Glam Nails
con WhatsApp automático

Cuando alguien reserva una cita en tu página, n8n le envía un mensaje de confirmación automáticamente.

🤖 ¿Qué es n8n?

Es una herramienta que conecta aplicaciones automáticamente. Por ejemplo: cuando llega una cita → envía un WhatsApp → guarda en Google Sheets → te notifica a ti también.

✅ Gratis si lo instalas tú

💰 ¿Cuánto cuesta?

Gratis: Si instalas n8n en tu PC o en un servidor propio (requiere dejarlo encendido).

De paga: n8n Cloud desde $20/mes si quieres que funcione sin tener tu PC encendida.

Cloud: ~$20/mes

📋 Cómo instalarlo gratis

1
Instala Node.js (si no lo tienes)
Necesario para correr n8n en tu PC

Ve a nodejs.org, descarga la versión LTS (botón verde) e instálala normalmente.

💡Si ya tienes Node.js instalado (el que usaste antes para NegocioApp no aplica — ese era .NET). Verifica escribiendo node --version en la terminal.
2
Instala n8n en tu PC
Un solo comando en la terminal

Abre una terminal (cmd o PowerShell) y escribe:

cmd
C:\> npm install -g n8n
# Espera ~2 minutos...
✓ n8n instalado correctamente
3
Arranca n8n
Cada vez que quieras usar la automatización
cmd
C:\> n8n start
Initializing n8n...
✓ Editor is now accessible via:
http://localhost:5678

Abre tu navegador y ve a http://localhost:5678 — ahí verás el panel de n8n.

4
Crea el flujo de automatización
Dentro del panel de n8n

En el panel de n8n, crea un nuevo Workflow con estos pasos:

Flujo: Confirmación de Cita

1
Webhook (Trigger)

Recibe los datos del formulario. Copia la URL que genera (algo como http://localhost:5678/webhook/glam-nails-cita)

2
WhatsApp (via Twilio o CallMeBot)

Envía mensaje al teléfono del cliente confirmando su cita. Usa el campo {{$json.mensaje_whatsapp}}

3
Google Sheets (opcional)

Guarda los datos de la cita automáticamente en una hoja de cálculo de Google.

4
Email / Telegram (opcional)

Te notifica a ti también cuando llega una nueva cita.

⚠️Para WhatsApp necesitas una API. La más fácil y gratis para empezar es CallMeBot (manda mensajes a tu propio WhatsApp) o Twilio (manda a cualquier número, tiene plan gratis).
5
Conecta la página web con n8n
Pega la URL del webhook en el archivo HTML

Abre el archivo glam-nails-con-n8n.html en VS Code y busca esta línea:

const N8N_WEBHOOK_URL = 'https://TU-N8N.com/webhook/glam-nails-cita';

Reemplaza esa URL por la que te genera n8n al crear el nodo Webhook. Por ejemplo:

const N8N_WEBHOOK_URL = 'http://localhost:5678/webhook/glam-nails-cita'; // Si usas n8n Cloud sería algo como: // 'https://tu-nombre.app.n8n.cloud/webhook/glam-nails-cita'
✅¡Listo! Cada vez que alguien llene el formulario, n8n recibirá los datos y enviará los mensajes automáticamente.
